## Development

### Requirements

- PHP 7.4+
- Composer 2.0.0+

### Local configuration

After cloning the repository install packages:

```commandline
composer install
```

To run server locally:

```commandline
./script/server
```

To run tests:

```commandline
./script/test
```

To run code checks and dependencies security scan:

```commandline
./script/lint
```
